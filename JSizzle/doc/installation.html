installation instructions

  1. Download and install JCurry
  2. Copy jsizzle.jar into your eclipse home folder
  3. Open up eclipse.ini in the same folder, and edit the line 
    -Xbootclasspath/a:lombok.jar;jcurry.jar
    to say 
    -Xbootclasspath/a:lombok.jar;jcurry.jar;jsizzle.jar
  4. Reference jsizzle.jar, jcurry.jar and guava-XXX.jar in your projects
  5. Enjoy modelling in Java
